# DIGIY HUB

## DIGIYLYFE — Le Club Entreprises du Terrain

DIGIYLYFE n’est pas une application de plus.

C’est une **Architecture Métiers Pro** : une structure qui relie les professionnels du terrain, leurs portes métiers, leurs fiches, leurs QR, leur voix, leur PAY, leur cockpit et leur admin.

La voix est la porte d’entrée : le public parle, DIGIY oriente, les bonnes portes remontent, et le professionnel garde la main.

**La voix ouvre.
L’Oreille comprend.
Les portes répondent.
PAY trace.
L’admin protège.
Le pro décide.**

---

## 👂 Oreille DIGIY

L’Oreille DIGIY est le moteur maison d’intention terrain de DIGIYLYFE.

Elle capte une demande parlée ou écrite, reconnaît l’intention, oriente vers les bonnes portes métiers et prépare l’action.

Elle ne décide jamais à la place du professionnel.

* [OREILLE_DIGIY_V1.md](./OREILLE_DIGIY_V1.md) — moteur maison d’intention terrain.
* [OREILLE_DIGIY_V2.md](./OREILLE_DIGIY_V2.md) — moteur maison + renfort IA locale contrôlé.

## Doctrine

DIGIY ne remplace pas les professionnels.

DIGIY leur donne des bras numériques :

* montrer ;
* organiser ;
* orienter ;
* partager ;
* préparer ;
* tracer ;
* protéger la relation ;
* garder la décision dans les mains du pro.

Le professionnel garde son nom, son client, son argent et sa décision.
